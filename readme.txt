Block or Delete spam user on Registration by email domain &amp; username.

== Description ==

So you are tired to fight with spam registration ? just install and activate and save the settings to running fight against spammy &amp; temporary email registration.

### Registration Control by http://pickplugins.com

<strong>Plugin Features</strong>

* Block or Delete user on registration.
* Block by email domain.
* Block by username match.
* List of activity.
* Notification email after block or delete user.
* Email template editor.

<strong>Translation</strong>

Pluign is translation ready , please find the 'en.po' for default translation file under 'languages' folder and add your own translation. you can also contribute in translation, please contact us http://www.pickplugins.com/contact/

== Installation ==

1. Install as regular WordPress plugin.
2. Go your plugin setting via WordPress Dashboard and find "<strong>Registration Control</strong>" activate it.
